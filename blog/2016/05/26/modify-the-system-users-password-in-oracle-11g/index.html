重置oracle 11g中sys用户的密码

May 26th, 2016 12:41 pm

重置oracle 11g中sys用户的密码的步骤如下：
1. 登录oracle服务器，并启动命令提示符窗口
2. 输入'sqlplus / as sysdba'，通过服务器认证连接oracle
3. 输入修改密码的指令'alter user sys identified by password'，其中password为修改后的密码。
